# 使用Git Clone本仓库

首先，请点击[点击此处](https://git-scm.com/)来下载最新版Git。安装完后在`命令提示符`中执行以下命令：

```bash
git --version
```

最终结果应该显示你当前安装Git版本的版本号，结果如下图所示：

![](D:\Documents\Github\ezg\1.png)

如果正确安装Git，你可以执行以下命令进行克隆

```bash
 git clone https://github.com/WinMinecraft/Easy-Girls.git
```
